# 🚀 QUICKSTART - Dashboard MIS ISLAMIYAH PARIAMAN

Mulai menggunakan dashboard dalam 5 menit!

## ⚡ Startup Cepat (5 Menit)

### 1. Buka Terminal
```bash
cd /workspaces/WEBSITE-MIS-ISLAMIYAH-PARIAMAN
```

### 2. Jalankan Server
**Pilih Salah Satu:**

#### Option A: Python (Recommended)
```bash
python -m http.server 8000
```

#### Option B: Node.js
```bash
npx http-server -p 8000
```

#### Option C: VS Code Live Server
- Right-click `index.html`
- Pilih "Open with Live Server"

### 3. Buka Browser
```
http://localhost:8000
```

### 4. Selamat! 🎉
Dashboard sudah bisa digunakan!
