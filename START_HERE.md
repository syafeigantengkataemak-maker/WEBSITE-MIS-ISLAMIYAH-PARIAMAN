- **Python 3.7+** (to run locally) - OR use any static web server
- **~10MB storage** (for local data)

## 🆘 COMMON QUESTIONS

### Q: Do I need a server?
**A**: No! Everything works on your browser with local storage. Runs offline.

### Q: Where does data get saved?
**A**: In your browser's local storage. Stays there until you clear cache.

### Q: Can I print reports?
**A**: Yes! Click "Cetak" button → Print preview → Print or save as PDF

### Q: Is this mobile-friendly?
**A**: 100%! Works great on phones, tablets, and desktops.

### Q: Can I share this with others?
**A**: Deploy to GitHub Pages, Netlify, Vercel (5 min setup) - see DEPLOYMENT_CHECKLIST

### Q: How do I add more students?
**A**: Use the form in the dashboard. Each student added is saved automatically.

### Q: Is it secure?
**A**: For v1.0 with local storage - yes, it's safe. For production, add authentication (see roadmap).

## 📞 NEED HELP?

### Most Common Issues

**"Styles not loading"**
→ Clear browser cache (Ctrl+Shift+Del)

**"Form won't submit"**
→ Fill all required fields, check validation messages

**"Data disappeared"**
→ Check if cleared browser cache, data is in LocalStorage

**"Print not working"**
→ Check popup blocker, try different browser

**"Mobile menu not showing"**
→ Refresh page (F5), or check browser zoom
